# BrightTV

## BrightTV Viewership Analytics

### Project Overview

This project analyses BrightTV user profiles and viewership transaction data to provide insights that will help the company grow its subscription base and improve customer consumption. The analysis focuses on user behaviour, viewing trends, content performance, and business recommendations for increasing engagement and subscriptions.

The project includes data cleaning, SQL analysis, Excel dashboards, and a final presentation summarising insights and recommendations.

## How the project Was Done

The project was completed in the following phases:

1. **Project Planning**

   * Defined business questions
   * Created flowchart and project timeline
   * Planned data analysis approach

2. **Data Processing**

   * Cleaned raw data
   * Converted dates from UTC to South African time
   * Created new fields (year, month, day, hour, time buckets)
   * Aggregated data for analysis

3. **Data Analysis**

   * User growth trends
   * Viewing patterns by day and time
   * Consumption by content type
   * Store/location and revenue analysis
   * Peak vs low consumption periods

4. **Dashboard & Visualisation**

   * Created Excel pivots and charts
   * Built dashboard showing key metrics
   * Prepared presentation slides

## Insights Found

Some key insights discovered from the analysis include:

* Viewing activity is higher during evenings and weekends.
* Some days and time periods have very low consumption.
* Certain content categories drive higher engagement than others.
* User activity trends can help determine when to release new content.
* Increasing content during low consumption periods may increase overall usage.

## Recommendations

To grow BrightTV’s user base and consumption:

* Release new content during low consumption days.
* Promote popular content categories.
* Offer promotions or free content during low usage periods.
* Improve personalised recommendations.
* Introduce loyalty or subscription incentives.
* Increase marketing campaigns targeting inactive users.
